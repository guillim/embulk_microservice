SSHKEY=default_env_SSHKEY (usually a string like : key - needs to be in the .ssh folder)
TUNNEL_HOST=default_env_TUNNEL_HOST (usually something like : ssh_username@34.119.45.10)
LOCAL_PORT=default_env_LOCAL_PORT (usually a number like : 4001 it needs to match the port in configuration.yml)
REMOTE_HOST=default_env_REMOTE_HOST (usually 127.0.0.1 except if we connect to some intranet)
REMOTE_PORT=default_env_REMOTE_PORT (usually a number like : 27017 depednding on the database you are connecting to)

SSHKEY2=default_env_SSHKEY2 (ex : key2 - but could be the same one as previously : key)
TUNNEL_HOST2=default_env_TUNNEL_HOST2 (ex : ssh_username2@312.96.1.7 but could be the same remote server - ssh_username@34.119.45.10)
LOCAL_PORT2=default_env_LOCAL_PORT2 (ex : 4002 )
REMOTE_HOST2=default_env_REMOTE_HOST2 (most of the time : 127.0.0.1)
REMOTE_PORT2=default_env_REMOTE_PORT2 (usually a string like : 5432 depending on the database you are connecting to)
